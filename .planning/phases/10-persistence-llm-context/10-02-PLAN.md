| multipart/form-data → route handler | Conteúdo do arquivo (bytes) chega sem sanitização prévia; magic bytes validados pelo dispatcher Phase 9 |
| route handler → getUserEntitlement | Pro-gate deve ocorrer antes de qualquer I/O de arquivo para prevenir bypass |
| attachmentContext → buildToolContextMessages | Conteúdo do documento pode conter instruções maliciosas; delimitadores em context-messages mitigam |
