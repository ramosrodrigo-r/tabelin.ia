<objective>
Implementar a infraestrutura de backend para o protocolo de mutação e Q&A. Isso inclui tradução de fórmulas (EN <-> pt-BR), enriquecimento de prompt com o contexto completo da planilha, roteamento real via OpenAI Structured Outputs / Streaming, e o mock de fixture local para execução sem chave.

**Mudanças concretas:**
1. Criar o tradutor de fórmulas `apps/web/src/server/ai/formula-translator.ts` que inverte o dicionário `PT_BR_TO_EN` para traduzir EN -> pt-BR e pt-BR -> EN de forma robusta (separador `;` vs `,`).
2. Adicionar testes unitários para o tradutor em `apps/web/tests/formula-translator.test.ts`.
3. Atualizar a rota `apps/web/src/app/api/chat/unified/route.ts` para receber `specOverride`, alimentar o prompt do modelo com o contexto da planilha, chamar a API da OpenAI (estruturada para mutação, streaming textual para Q&A), traduzir as fórmulas resultantes da mutação de EN para pt-BR, e enviar os eventos NDJSON no stream.
4. Integrar o provedor de fixture local determinístico na rota se `OPENAI_API_KEY` não estiver definida.
5. Ajustar os testes de rota em `apps/web/tests/unified-route.test.ts` para alinhar com os novos comportamentos.
</objective>

    1. Criar `apps/web/src/server/ai/formula-translator.ts`:
       - Importar `PT_BR_TO_EN` de `@tabelin/shared`.
       - Inverter o mapa para criar `EN_TO_PT_BR`.
       - Implementar `translateEnToPtBr(formula: string): string`:
         - Substituir nomes de funções detectadas antes de `(` pelo correspondente pt-BR.
         - Substituir separadores de argumento `,` por `;` caso esteja dentro de parênteses de função e fora de aspas duplas de strings.
       - Implementar `translatePtBrToEn(formula: string): string` de forma análoga (útil para normalizar entradas de usuário).
    2. Criar `apps/web/tests/formula-translator.test.ts` com testes cobrindo:
       - Tradução de nomes de funções (ex.: `SUM` -> `SOMA`, `IF` -> `SE`, `VLOOKUP` -> `PROCV`).
       - Tradução de argumentos separadores (ex.: `SUM(A1, B1)` -> `SOMA(A1; B1)`).
       - Preservação de decimais e textos (ex.: `IF(A1 > 3.5, "A, B", "C")` -> `SE(A1 > 3.5; "A, B"; "C")`).
