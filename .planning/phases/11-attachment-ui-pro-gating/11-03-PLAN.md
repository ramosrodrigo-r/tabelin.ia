## Trust Boundaries

| Boundary | Descrição |
|----------|-----------|
| user file input → handleFileSelect | Arquivo do filesystem do usuário — validado por validateFile antes de setar state |
| drag-and-drop → handleFileSelect | Arquivo arrastado — mesma validação; ignorado se !isPro |
| FormData → fetch /api/tools/formula/generate | Content-Type gerenciado pelo browser (boundary automático) |
| attachment_grounded.extractedText → FormulaOutputPanel | Texto do documento do usuário — renderizado como texto puro pelo AttachmentPanel (11-02) |
