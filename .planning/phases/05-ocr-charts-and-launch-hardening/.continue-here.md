## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| 127.0.0.1 vs localhost mismatch | BETTER_AUTH_URL=localhost:3000 but Playwright used 127.0.0.1:3000 → CSRF origin validation rejected all auth requests (403) | advisory | playwright.config.ts baseURL must match BETTER_AUTH_URL exactly (now fixed to localhost:3000) |
| page.request vs page.evaluate for auth | page.request.post() does not share cookie jar with page context → session cookie not visible in subsequent page.goto() | advisory | Always use page.evaluate(async () => fetch(...)) for auth API calls in Playwright tests |
| React not hydrating in dev-server headless | No __reactFiber keys on DOM elements even after networkidle+3s. Native form submit (GET) fires before React onSubmit. | advisory | Use API-based auth (page.evaluate fetch). Do NOT rely on form UI interaction for smoke test setup. |
