# Phase 05 Plan 04: Gap Closure — OCR Fixture Fallback + Chart Fixture

## Goal

Corrigir os 2 issues encontrados no UAT da Fase 05 que impedem OCR e Charts de funcionarem em modo dev (sem OPENAI_API_KEY).

## Gap Summary

| Gap | Root Cause | Severidade |
|-----|-----------|-----------|
| OCR retorna 502 sem API key | `createOpenAIClient()` lança exceção; `OCR_FIXTURE_RESPONSE` nunca importado em `ocr-processor.ts` | major |
| Chart não renderiza sem API key | `createFixtureStream` retorna texto descritivo; não detecta requisição de gráfico nem usa `chartDataFixture` | major |

## Threat Model

| ID | Ameaça | Mitigação |
|----|--------|-----------|
| T-05-04-01 | Fixture expõe dados reais de produção | `chartDataFixture` e `OCR_FIXTURE_RESPONSE` contêm apenas dados sintéticos hardcoded — nenhum dado de usuário |
| T-05-04-02 | Detecção de chart por substring pode ser frágil | Usar substring específica do `CHART_PROMPT` (`"chartType"`) — string única e controlada internamente |

### Task 2 — Charts: detectar CHART_PROMPT em `createFixtureStream` e retornar `chartDataFixture`

**Arquivo:** `apps/web/src/server/ai/file-chat-stream.ts`

**Mudança:**
1. Adicionar `userMessage: string` como segundo parâmetro de `createFixtureStream`
2. Importar `chartDataFixture` de `@tabelin/shared`
3. Detectar requisição de gráfico por `userMessage.includes("chartType")` (substring única do CHART_PROMPT)
4. Se chart request: content = `JSON.stringify(chartDataFixture)`; senão: conteúdo descritivo atual
5. Atualizar chamada em `buildFileChatStream` passando `userMessage`

```typescript
// Import adicional
import { chartDataFixture } from "@tabelin/shared";

// createFixtureStream atualizada
function createFixtureStream(
  schema: FileSchema,
  userMessage: string,
  lastFreeUse?: boolean
): ReadableStream<Uint8Array> {
  const isChartRequest = userMessage.includes("chartType");

  const content = isChartRequest
    ? JSON.stringify(chartDataFixture)
    : `Arquivo analisado (modo fixture). Schema detectado: ${schema.columns.length} colunas — ${schema.columns.map((c) => c.name).join(", ")}.`;

  // resto inalterado...
}

// buildFileChatStream: passar userMessage
if (!process.env.OPENAI_API_KEY) {
  return createFixtureStream(schema, userMessage, lastFreeUse);
}
```

**Acceptance criteria:**
- "Sugerir Gráfico" sem API key → ChartMessage renderizado com dados de `chartDataFixture`
- Pivot/Relatorio sem API key → resposta de texto descritiva (comportamento atual)
- TypeScript: 0 erros

**Commit:** `fix(05-04): detect CHART_PROMPT in fixture stream and return chartDataFixture`
