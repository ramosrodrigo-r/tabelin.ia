# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-05-26
**Phases:** 1–5
**Total Plans:** 16

## Overview

Tabelin.IA v1.0 MVP — Brazilian SaaS for spreadsheet and data productivity. Delivers a full end-to-end product: authenticated workspace, localized formula generation, freemium billing with Mercado Pago, multi-tool suite (scripts, SQL, regex), CSV/XLSX file analysis with AI chat, OCR from table images, chart rendering, and a full Playwright smoke test suite covering all MVP happy paths.

## Phases

### Phase 1: Localized Formula Workspace

**Goal**: Users can sign in and use a polished Brazilian formula assistant that returns copy-ready localized formulas and explanations.
**Depends on**: Nothing (first phase)
**Plans**: 3 plans

Plans:
- [x] 01-01: Scaffold Next.js workspace, auth, sessions, and navigation shell.
- [x] 01-02: Implement formula/explainer tool contracts, locale/platform selectors, prompt builders, and structured validation.
- [x] 01-03: Implement streaming output UI, copy feedback, assumptions/warnings, and formula MVP verification.

**Details:**
Full Next.js 16 / React 19 / TypeScript app scaffolded with pnpm workspace. Better Auth with email/password, signed httpOnly session cookie, Prisma 7 schema (User, Session, Account). Authenticated sidebar workspace shell. Formula tool with Brazilian locale (semicolon separators), platform selector (Excel/Sheets/Airtable/LibreOffice), generate + explain modes, streaming output UI, copy-ready button with feedback, assumptions/warnings display. Playwright MVP path verified.

### Phase 2: Freemium Billing and Entitlements

**Goal**: Free and Pro plan behavior works end to end with strict quotas and webhook-driven billing state.
**Depends on**: Phase 1
**Plans**: 3 plans

Plans:
- [x] 02-01: Build transactional usage ledger, quota reservations, and free-tier enforcement.
- [x] 02-02: Integrate Mercado Pago checkout and webhook reconciliation.

## Milestone Summary

**Key Decisions:**

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Brazil-first rather than generic spreadsheet AI | Localization is the main competitive advantage | ✓ Validated |
| Vertical MVP phase structure | End-to-end slices usable and demoable at each phase | ✓ Validated across all 5 phases |
| Include auth + quotas + payments in v1 | Monetization enforcement is a core MVP constraint | ✓ Validated |
| Mercado Pago Checkout Pro for billing | Brazilian Pix/card without multi-bank complexity | ✓ Validated |
| Small upload limits at launch (5 MB) | Controls cost, latency, parsing complexity, data risk | ✓ Validated |
| Commercial LLM APIs with privacy controls | Avoids custom model training; OpenAI Vision for OCR | ✓ Validated |
| Fixture fallback when OPENAI_API_KEY absent | Enables dev/test without real API costs | ✓ Validated |
| Mock AI/billing in E2E; real auth/DB | Isolates flaky external calls, keeps auth/quota paths real | ✓ Validated — 9/9 smoke tests pass |
