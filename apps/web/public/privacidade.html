Política de Privacidade de Dados

Esta página descreve como o Tabelin.IA trata os dados dos usuários, em especial os arquivos enviados para análise.

Dados de Planilhas Enviados

Quando você faz upload de um arquivo .csv ou .xlsx para a ferramenta de Análise de Planilhas:

  • O arquivo é processado inteiramente em memória no servidor. O conteúdo bruto (binário) é descartado imediatamente após a extração dos dados estruturados.
  • O conteúdo raw do arquivo nunca é armazenado em disco nem no banco de dados.
  • Apenas metadados estruturados — nomes de colunas, tipos de dados inferidos e amostras representativas de linhas — são persistidos temporariamente no banco de dados para possibilitar o chat conversacional.

Uso de Inteligência Artificial

Os metadados estruturados extraídos do seu arquivo são enviados à API Comercial da OpenAI para análise e geração de respostas. O conteúdo bruto do arquivo nunca é enviado ao provedor de IA.

De acordo com os Termos de Uso da API Comercial da OpenAI, dados enviados via API não são utilizados para treinar os modelos públicos da OpenAI, a menos que o cliente opte explicitamente por compartilhá-los. Tabelin.IA não opta por compartilhamento de dados para treinamento.

Retenção e Deleção Automática

Registros de análise de arquivos são automaticamente apagados após 1 hora de inatividade. A deleção ocorre de forma independente de qualquer ação do usuário, por meio de um processo automático periódico. O histórico de mensagens de chat associado ao arquivo também é apagado em cascata.

Logs de Aplicação

O conteúdo de células do arquivo não é registrado em logs de aplicação. Apenas metadados de operação (como tamanho do arquivo, tipo e número de linhas processadas) podem aparecer em registros internos para fins de monitoramento.

Seus Direitos

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), você tem direito a solicitar informações sobre seus dados, sua correção ou eliminação. Para exercer esses direitos, entre em contato pelo e-mail abaixo.
